## Blog Setup

This portfolio includes a blog feature powered by Strapi CMS. To use it:

1. Set up a Strapi instance with an `articles` content type
2. Create a `.env` file (see `.env.example` for reference)
3. Configure your Strapi URL and API token:
   ```bash
   VITE_STRAPI_URL=https://your-strapi-instance.com
   VITE_STRAPI_API_TOKEN=your-api-token-here  # Optional
   ```
4. The blog will automatically fetch and display articles from your Strapi instance

### Strapi Content Type Structure

Your Strapi `articles` collection should have these fields:
- `title` (Text)
- `content` (Rich Text)
- `description` (Text, optional)
- `slug` (UID)
- `cover` (Media, optional)
- `author` (Relation, optional)
- `publishedAt` (DateTime)
